Schema  Model

Using Model we do CRUD operation

explaination/working:
1. Schema - define the structure of the database
2. Model - define the structure of the database and also do CRUD operation
Schema: Defines the structure of the database. It outlines how data is organized and the relationships between different parts of the data.
Model: Represents the schema and is used for performing CRUD (Create, Read, Update, Delete) operations on the data.

In summary, the schema defines the structure of the data, while the model utilizes this structure to perform various operations on the data.
